# Gopeed

--------------------

Gopeed（全称 Go Speed），直译过来中文名叫做够快下载器（不是狗屁下载器！），是一款由Golang+Flutter开发的高速下载器，支持（HTTP、BitTorrent、Magnet）协议下载，并且支持全平台使用。除了基本的下载功能外，Gopeed 还是一款高度可定制化的下载器，支持通过对接APIs或者安装和开发扩展来实现更多的功能。
