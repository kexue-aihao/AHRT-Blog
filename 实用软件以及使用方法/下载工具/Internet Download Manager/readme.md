# Internet Download Manager是什么？

------------------------------

Internet Download Manager是一个用于Windows系统的下载管理器。 IDM可以让用户自动下载某些类型的文件，可将文件划分为多个下载点以更快下载，并行出最近的下载，方便访问文件。相对于其他下载管理器，具有独特的动态文件分割技术。

IDM最高下载线程可以达到32，是一个非常好用的多线程下载工具，还可以添加文件类型的后缀
